# Package Manager Builders

[![CI](https://github.com/advent06564/packagemanger-builders/actions/workflows/ci.yml/badge.svg)](https://github.com/advent06564/packagemanger-builders/actions/workflows/ci.yml)

Windows Package Manager Installation Script (2026 Edition). Installs the four major Windows package managers automatically.

## Package Managers Installed

| Manager | Description |
|---------|-------------|
| **WinGet** | Pre-installed on Win 10/11; verifies registration |
| **Chocolatey** | Veteran community package manager |
| **Scoop** | Portable developer tools |
| **PowerShellGet** | Native module manager |

## Getting Started

### Prerequisites

- Windows 10/11
- Python 3.x
- **Administrative Terminal** (required)

### Run

```powershell
python install_managers.py
```

> ⚠️ Must be run from an Administrative Terminal.
